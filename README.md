THIS METHOD CAN BE USED ON SO MANY THINGS LIKE UNPKG AND JSDELIVR/NPM 

1. open/create a empty folder
2. create a npm account and turn on 2fa
3. run `npm adduser` in your terminal then add your user in the browser
4. run `git clone https://github.com/genizy/npm.git .`in your terminal 
5. open RUN.py and edit `tuffname` to a package that does NOT exist, otherwise it will require your username in front
6. run RUN.py after editing it
7. run `npm publish`in your terminal 
8. the site should be usable at `https://unpkg.com/<NAME>/index.svg`
